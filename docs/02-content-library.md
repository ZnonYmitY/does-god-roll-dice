# 上帝掷骰子吗？｜文字库与模板设计 v1.0

## 1. 文案原则

### 1.1 目标风格

- 一本正经地处理鸡毛蒜皮；
- 黑色幽默、存在主义、自嘲和失败美学；
- 技能之间互相打断、否定或误解；
- 文案短、可截图、可独立传播；
- 不追求“正确建议”，追求“像一个脑内技能真的说了这句话”。

### 1.2 禁止风格

- 鸡汤；
- AI 助手式总结；
- 过度礼貌；
- 真实专业建议；
- 长篇解释；
- 直接复制原作台词；
- 只替换名词、语义不通的模板。

### 1.3 常用句法

- `{object}不是答案。{object}是答案还没学会说话之前的样子。`
- `你不是想要{object}。你是想要一个暂时替你承担存在压力的东西。`
- `这不是失败。这是系统以一种不体面的方式保护了你。`
- `命运没有回答。命运只是把骰子推了回来。`
- `你可以这么做。但不要假装这是理性。`

## 5. 对象提取

### 5.1 句式

优先识别：

```text
要不要 + 动作 + 对象
想不想 + 动作 + 对象
该不该 + 动作 + 对象
能不能 + 动作 + 对象
是不是该 + 动作 + 对象
动词 + 不 + 动词 + 对象
```

### 5.2 动作词

```text
吃、喝、点、买、去、发、回、见、写、做、开始、继续、放弃、睡、休息、学习、上班、辞职
```

### 5.3 非问句

- `今晚炸鸡` → 炸鸡
- `不想上班` → 上班
- `发朋友圈` → 朋友圈
- `辞职算了` → 辞职
- `猫在看我` → 猫

### 5.4 语义约束

通用模板不得无条件套用。例如：

> `{object}不是答案。{object}是答案还没学会说话之前的样子。`

适合：炸鸡、奶茶、睡觉、辞职等欲望或行动对象。  
不适合：朋友、妈妈、疾病、某个具体人物姓名。

每个模板应包含：

```json
{
  "allowedCategories": ["food", "shopping", "rest"],
  "blockedEntityTypes": ["person", "medical", "safety"]
}
```

## 7. 核心技能语气与示例

### 7.1 Electrochemistry / 电化学

角色：欲望、食物、消费、即时反馈。

- `{object}不是答案。{object}是答案还没学会说话之前的样子。`
- 别再假装这是一个高层次问题了。身体已经投票，票上写着：`{object}`。
- 你需要的不是解释。你需要一点即时反馈，最好带糖、盐、热量或错误。
- 理性会明天再来。今晚先让多巴胺坐在驾驶座上。
- `{object}`正在发光。不是道德意义上的光，是欲望意义上的。

### 7.2 Logic / 逻辑思维

角色：成本、收益、拆穿借口。

- 根据目前可见信息，这不是最理性的选择。但你显然不是来这里寻找理性的。
- 证据不足，冲动过量，结论可疑。
- 你已经知道答案了。你只是希望骰子替你承担责任。
- 从成本、收益和明天的后悔概率来看，这个方案有明显漏洞。
- 逻辑不反对快乐。逻辑只反对你把快乐包装成必然。

### 7.3 Volition / 意志力

角色：坚持、自我保护、保持完整。

- 站起来。不需要英勇，只需要离开当前这个姿势。
- 你可以失败。但不要在还没开始前就替失败铺好床。
- 这不是命运。这只是你和惯性之间的一次小型扭打。
- 你不需要变强。你只需要先做一件不会让自己更虚弱的事。
- 别把所有选择都交给疲惫。疲惫不是君主。

### 7.4 Inland Empire / 内陆帝国

角色：预感、梦、荒诞和已读乱回。

- `{object}`在看着你。不是用眼睛，是用某种更旧、更潮湿的东西。
- 这件事不是选择。它是一扇门，门后有人穿着你的旧记忆。
- 你听见了吗？不是声音。是某种还没成为声音的预感。
- 不要问为什么。问就是门已经开了一条缝。
- 你以为你输入的是问题。其实你输入的是召唤词。

### 7.5 Shivers / 寒颤

角色：城市、天气、远方和宏观旁白。

- 城市在你窗外缓慢呼吸。它不关心你的选择，但记得每一次犹豫。
- 楼下有风。风里有车灯、外卖箱和一些没被说出口的话。
- 某个路口的红灯亮了。与此同时，你的脑内也有东西停住了。
- 城市的皮肤上爬过一阵冷意。它像是在等你做出一点微不足道的决定。
- 远处有人关上一扇门。你不知道为什么这和你有关，但它就是有关。

### 7.6 Drama / 戏剧

角色：表演、夸张、不可靠叙事。

- 当然要这么做。否则这一幕的戏剧张力从何而来？
- 你不是在做选择。你是在给自己的传记增加一段可疑但精彩的脚注。
- 你需要的不是答案，是一个能让你显得更复杂的理由。
- 如果要崩溃，至少崩溃得有层次。
- 这一刻需要灯光、沉默，以及一个不完全负责任的决定。

### 7.7 Composure / 镇定

角色：体面、社交面具、保持表面稳定。

- 脸可以保持平静。问题是，里面那部分已经开始搬家具了。
- 你可以装作没事。这项技能你练得不错，甚至有点令人担忧。
- 先把表情放回原位。其余部分稍后处理。
- 体面不是答案。但它可以暂时充当一把椅子。
- 不要让他们看出来。也不要让自己完全相信自己没事。

### 7.8 Conceptualization / 概念化

角色：创作、意义和强行升华。

- 这不是拖延。这是尚未被命名的前期概念孵化。
- 你可以把这件事包装成一个项目。包装之后，它至少不像逃避。
- 一个想法正在成形。它很丑，但它活着。
- 意义不是被发现的。意义是你硬给它套上的标题。
- 如果这件事失败了，你仍然可以称它为一次实验。

## 9. 最终判定示例

### food

**大失败**

- 别吃。不是因为你自律，而是因为连骰子都看不下去这场欲望政变。
- 今晚撤退。`{object}`没有输，它只是被命运暂时扣押了。

**失败**

- 先别吃。不是永别，只是今晚这份欲望需要冷静一下。
- `{object}`可以等。你的钱包、胃和明天的精神状态都请求发言。

**混沌成功**

- 吃吧。但别说这是胜利。今晚这更像一次停火。
- `{object}`通过了检定，但方式不太光彩。你可以接受这份不光彩。

**成功**

- 吃吧。今晚你的灵魂需要的不是解释，而是一点热量和短暂的和平。
- `{object}`获得通过。不是因为它正确，而是因为它足够及时。

**大成功**

- 大成功。`{object}`不再是选择，它是神谕，是油脂写成的启示。
- 两枚骰子同时点头。今晚，`{object}`被批准成为你的精神急救包。

### social

**大失败**

- 不去。今天门外不是世界，是一台会消耗你全部电量的机器。
- 大失败。你的社交系统拒绝启动，并发出一声很小但很诚实的悲鸣。

**失败**

- 今天不适合出现。你可以把自己暂时归还给房间、热水和沉默。
- 你的脸也许能微笑，但你的灵魂明显没有预约这场活动。

**混沌成功**

- 可以去。但不要试图成为派对之王。你今天的任务是像家具一样安静地存在。
- 去吧，但设置撤退路线。每个体面的人都应该有逃生通道。

**成功**

- 去。你不一定会快乐，但可能会捡到一点久违的人类连接。
- 把自己带出去，哪怕只是短暂地被世界重新加载。

**大成功**

- 出门。命运已经把鞋摆到了门口。
- 今晚你甚至可能在人群里找回一点信号。

### work_study

**大失败**

- 任务没有被完成，甚至开始怀疑自己是否真实存在。
- 别硬撑了。你现在不是低效，是系统正在冒烟。

**失败**

- 先做最小的一步。不要幻想胜利，先避免彻底投降。
- 今天不适合宏大计划。适合打开文件，然后盯着它看三分钟。

**混沌成功**

- 可以开始。但请把目标缩小到不会吓死自己的程度。
- 打开它。不要评价，不要规划，不要宣誓。只是打开。

**成功**

- 开始。不是因为你准备好了，而是因为再不开始它就要长牙了。
- 你不需要热血，只需要把第一个字敲出来。

**大成功**

- 开始吧。今天的意志力罕见地穿上了鞋。
- 命运给了你一小段清醒窗口，不要浪费。

### absurd

**大失败**

- 系统试图理解你，结果理解了自己不该理解的东西。
- 骰子掉进了桌缝。桌缝拒绝归还。

**失败**

- 答案没有出现，但一只形状可疑的念头坐到了旁边。
- 系统没有解决这个问题。系统只是把它放进了一个更暗的盒子。

**混沌成功**

- `{object}`暂时被解释了，但解释本身看起来不太健康。
- 可以。也不可以。骰子给出了第三种更潮湿的可能。

**成功**

- `{object}`不是问题，是一封误投到你脑内的信。
- 系统决定承认这个问题的合法性，虽然非常不情愿。

**大成功**

- `{object}`正式进入你的精神档案，并获得一把小椅子。
- 两枚骰子同时停下。宇宙看了你一眼，没有否认。
